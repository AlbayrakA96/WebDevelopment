Opdracht 8: Wat is een strict equality comparison ?

== betekent dat het gelijk is aan de soort
=== staat gelijk aan soort en type 

            ==	equal to	x == 8	false	
                            x == 5	true	
                            x == "5"	true

            ===	equal value and equal type	x === 5	true	
                                            x === "5"	false
