JavaScript: Introductie Build in methods

Opdracht 16.

Maak een eigen voorbeeld van de volgende functies en licht toe wat de functie doet in het blok:

                        1. charAt()
                        2. concat()
                        3. slice()
                        4. split()
                        5. substr()

Een voorbeeld van de charAt() functie.

                        let voorNaam = "Abdusselam";
                        let eersteLetter = voorNaam.charAt(0);

                        console.log(eersteLetter);

Geeft een character uit de string terug.

Een voorbeeld van de concat() functie.

                        let voorNaam = "Abdusselam ";
                        let achterNaam = "Albayrak";

                        let namenAlbayrak = voorNaam.concat(achterNaam);

Met de functie concat() kan je meerdere strings aan elkaar toevoegen.

Een voorbeeld van de slice() functie.

                        let mercedesModel = "AMG GT63";
                        let typeMotor = mercedesModel.slice(0,3);\

                        console.log(typeMotor);

Met de functie slice() haal je een deel uit de string, en deze kan je terug roepen.

Een voorbeeld van de split() functie.

                        let text = "Hallo hoe gaat het met u?";
                        let splitSentence = text.split(" ");

                        console.log(splitSentence);

Met de functie split() verdeel je de string in array van substrings.

Een voorbeeld van de substr() functie.

                        let text = "The sky is no limit";
                        let substrLimit = text.substr(13, 18)

                        console.log(substrLimit);

The substr() method extracts parts of a string, beginning at the character at the specified position, and returns the specified number of characters.
