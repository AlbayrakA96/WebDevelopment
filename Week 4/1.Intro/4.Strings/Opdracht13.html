1. Maak een String variable aan met de waarde “3”.

                let text = "3";
                let number = 3;

                let combinedText = text + number;
                
                console.log(combinedText)

Wat merk je op?

Als een number en string datatype samen worden gevoegd, worden te datatypes aan elkaar gevoegd.
