JavaScript: Introductie Numbers

Opdracht 9.

1. Copy de volgende code in jouw VSCode:

                let number = 3;
                number++;

2. Print uitkomst in jouw browser console.

                console.log(number);

Console print het getal 4 uit.

3. Wijzig number++ naar number--

                let number = 3;
                number--;

4. Wat is de uitkomst van de som ?

                console.log(number);

Console print het getal 2 uit.

5. Copy de volgende code in jouw VSCode:

                let number = 3;
                const sum = number++ + 2;
                console.log(sum);

Q 6. Wat is de uitkomst van van de som ?

A. De uitkomst van de som is 5. Er is een let variable met de waarde 3, daar increment je 2 bij.
