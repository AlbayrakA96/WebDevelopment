Maak nu een Person object door gebruik te maken van Object initializers. Wat zou eventueel de voor en nadelen t.o.v Object constructors?

           const Person = {
               firstName:"Abdusselam", 
               lastName:"Albayrak", 
               age:24
            };

Wat zou eventueel de voor en nadelen t.o.v Object constructors?

Voordelen = Als je een kleine object gebruikt kan je beter een initializer gebruiken omdat het compacter eruit ziet. Alles wordt in 1 lijn gecodeerd.

Nadelen = Grote object met veel gegevens die je opnieuw moet gebruiken
