1. Copy de volgende code in jouw VSCode:

let number = 3; number += 4; console.log(number);

2. Wat is de uitkomst van van de som ?

De uitkomst is 7.

3. Hoe zou je het anders kunnen schrijven en dezelfde uitkomst hebben ?

let number = 3;
number = number + 4;
console.log(number);

Doordat number al een waarde heeft van 3, geef je aan dat number een nieuwe waarde moet krijgen door number + 4 te doen.
