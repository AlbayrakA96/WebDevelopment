Opdracht 8. cascade en inheritance

Leg in eigen woorden uit wat met cascade en inheritance wordt bedoeld. Maak voorbeelden om je antwoord uit te leggen.

Cascade

Bij Cascade gaat het om de volgorde die in je CSS staat; Wanneer je 2 dezelfde selectors heb met dezelfde waarde, wordt het onderste rule doorgevoerd.

Voorbeeld:
                h1 {
                    color: red;
                }

                h1 { 
                    color: blue;
                }

inheritance

Wanneer je een element target zoals body, wordt alles wat binnen de body valt ook aangesproken, behalve als je het apart target

Voorbeeld
                .test {
                        color: blue;
                    }
                    
                span {
                        color: red;
                    }
